0x01 整除

定义：若整数 n 除以整数 d 的余数为 0，即 d 能整除 n，则称 $d$ 是 $n$ 的约数，记为 $d \mid n$。

常用性质：

  • 性质1：$\begin{aligned} a \mid b,b \mid c \Rightarrow a \mid c \end{aligned}$

  • 性质2：$\begin{aligned} a \mid b \implies a \mid kb. \end{aligned}$

  • 性质3：$\begin{aligned} a \mid b, a \mid c \Rightarrow a \mid kb \pm lc, \end{aligned}$

  • 性质4：$\begin{aligned} 若 a \mid kb, 且a 与k互质, 则a \mid b \end{aligned}$

性质1告诉我们整除满足一种类似结合律的东西；

性质3是性质2的拓展，需要注意的是性质4告诉我们只有在特定条件下性质2的逆命题才会成立；

在实际题目中，经常会用性质3去发现新的整除

例题：$\begin{aligned} x, y, z均为整数, 若11 \mid (7x + 2y - 5z), 求证: 11 \mid (3x - 7y + 12z). \end{aligned}$

​ 考虑用性质3去证明，令A=(7x + 2y - 5z),B=(3x - 7y + 12z)，我们希望能凑出来一个C=kA+lB且系数被11整除的形式，发现当k=2,l=-1时，恰好满足。所以11整除C，又因为11整除A，所以可以根据性质3得出11整除B

0x02 素数

素数定理：设$\pi(x)$为1到x中素数的个数，由于精确公式较为繁琐，只列出一个极限发现

$$ \begin{aligned} \lim_{x\to\infty} \frac{\pi(x)}{\left(\frac{x}{\ln(x)}\right)} = 1 \end{aligned} $$

一些推论：

  • 相邻的两个数字一定互质

0x03 GCD

裴蜀（Bézout）定理：设 $a,b$ 是不全为0的整数，则存在无穷多组数对 $(x,y)$，满足不定方程 $ax+by=d$，其中 $d=gcd(a,b)$。

换句话说，二元一次方程的结果只会是ab两个系数gcd的整数倍，这个结论相当关键并且应该经常依靠这个解题

  • 常用的处理有：将模数视为 $y$，转化成不定方程求解问题。

特别地，若两个系数互质的话（即 $gcd(a,b)=1$），$ax+by$ 可以表示任意数字


从质因数分解角度理解GCD与LCM本质：

$$\begin{aligned} a = p_1^{x_1}p_2^{x_2}\ldots, \quad b = p_1^{y_1}p_2^{y_2}\ldots \end{aligned}$$$$a \times b = p_1^{x_1+y_1} p_2^{x_2+y_2} ...$$$$\begin{aligned} \text{GCD}(a, b) = p_1^{\min(x_1,y_1)} p_2^{\min(x_2,y_2)} \dots \end{aligned}$$$$\begin{aligned} a \times b/GCD(a,b) = p_1^{x_1+y_1-\min(x_1,y_1)} \dots = p_1^{\max(x_1,y_1)} \dots = LCM(a,b) \end{aligned}$$

辗转相减法求GCD:

​ 之所以先介绍辗转相减是感觉它更接近本质，假设求两数 $A,B$的gcd，我们可以将其分解为

$A=gcd(A,B)*a,B=gcd(A,B)*b$，由于两数相减，他们的公共因子不会消失，所以我们考虑一直做大减小，直到两数相等或较小数为0（与辗转相除终止条件对应，实际上不过是两数相等时多操作了一次使一个数变成0）。

我们发现当大数（感觉怎么说也要 $10^{100}$ 这种远超范围需要高精度的数据）取模运算会相当消耗时间（因为在高精度减法和取模复杂度差别巨大），我们考虑用优化的辗转相减法处理，具体的优化部分就是当出现含2的因子时直接除去，就能保证复杂度从O(n)降低到O(logn)了

辗转相除法求GCD:

​ 考虑我们有没有办法可以加速大减小这个过程，我们发现取模运算实际上是多次减法，直到一个数严格小于另一个数：$\begin{aligned} a \bmod b = a - \lfloor a/b \rfloor \cdot b \end{aligned}$ ，$gcd(a,b)=gcd(b,a \mod b)$

因此，它的终止条件就是较小数为0的时候。


以下是一些相关拓展结论

  • 推论1：对于方程 $ax+by=kd$ ，假设 $a=4,b=6$，会发现x每增加3，y需要维持方程成立就需要减少2，更普遍的来说 $x$ 每增加 $b/d$，$y$ 需要减少 $a/d$，这会在后续扩展欧几里得求方程通解中用到

  • 推论2：$\begin{aligned} \forall a, b, z \in \mathbb{N}^*, \gcd(a, b) = 1, \exists x, y \in \mathbb{N}, ax + by = ab - a - b + z \end{aligned}$，即两互质的数，不考虑负系数的情况下，表示不出来的最大的数是 $ab-a-b$

证明：待补，目前没找到特别严谨的证明

  • 推论3：一组数求 $LCM$，需要逐个像求 $GCD$ 一样，直接连乘再除以整个数列的 $GCD$是错误的，原因可以参考上边的本质理解
  • 推论4：$gcd(a,b)=d$ 可以化成 $gcd(a/d,b/d)=1$ 变成互质问题，这个推论相当重要
