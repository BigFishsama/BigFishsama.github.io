欧拉函数

#0x00 定义

设 $n$ 是一个正整数，欧拉函数 $phi(n)$ 定义为不超过 $n$ 且与 $n$ 互质的正整数的个数。

$$phi(n) = \sum_{i=1}^n [gcd(i,n)=1]$$

例题

P2398

https://www.luogu.com.cn/problem/P2398

$$ \sum_{i=1}^{n} \sum_{j=1}^{n} \gcd(i,j) $$

观察欧拉函数定义，我们希望将式子通过它化简，考虑将 $i$ 根据 $gcd$ 的值分类

$$ \sum_{d=1}^n \sum_{i=1}^n d \sum_{j=1}^n [gcd(i,j)=d] $$

  • 对于$[gcd(i,j)=d]$ 可以利用 $gcd$ 的性质，转化为 $[gcd(i/d,j/d)=1]$

  • 设 $k=i/d$ , $m=j/d$

    $$ \sum_{d=1}^n \sum_{k=1}^{n/d} d \sum_{m=1}^{n/d} [gcd(k,m)=1] $$

​ 这是用欧拉函数化简的极限，接下来的处理需要用到莫比乌斯反演

P2303

https://www.luogu.com.cn/problem/P2303

$$ \sum_{i=1}^n \gcd(i,n) $$

思路同上：

$$ \sum_{d|n} d\sum_{i=1}^n[gcd(i,n)=d] $$$$ \sum_{d|n} d\sum_{i=1}^{n}[gcd(i/d,n/d)=1] $$

令 $k=i/d$, $m=n/d$得：

$$ \sum_{d|n} d\sum_{k=1}^{m}[gcd(k,m)=1] $$$$ \sum_{d|n} d*phi(n/d) $$

#include <bits/stdc++.h>
#define int long long
#define debug(x) cout<<#x<<':'<<x<<' ';
#define N
using namespace std;
int T,m;
int getphi(int x)
{
    int res=x;
    for(int t=2;t<=sqrt(x);++t)
    {
        if(x%t==0)
        {
            res=res/t*(t-1);
            while(x%t==0) x/=t;
        }
    }
    if(x>1) res=res/x*(x-1);
    return res;
}
void solve()
{
    int n;
    int ans=0;
    cin>>n;
    for(int t=1;t<=sqrt(n);++t)
    {
        if(n%t==0)
        {
            if(t==n/t) 
            {
                ans+=getphi(n/t)*t;
            }
            else 
            {
                ans+=getphi(n/t)*t+getphi(t)*(n/t);
            }
        }
    }
    cout<<ans;
}
signed main()
{
    T=1;
    // cin>>T;
    while(T--)
    {
        solve();
    }
    return 0;
}

#include <bits/stdc++.h>
#define int long long
#define debug(x) cout<<#x<<':'<<x<<' ';
#define N 10000007
using namespace std;
int T,n,m;
int valid[N],prime[N],phi[N],sum[N];
int cnt=1;
void init()
{
    phi[1]=1;//必须加，不然会RE
    cnt=0;
    for(int t=2;t<=n;++t)
    {
        if(!valid[t])
        {
            valid[t]=1;
            phi[t]=t-1;
            prime[++cnt]=t;
        }
        for(int i=1;i<=cnt;++i)
        {
            if(t*prime[i]>n) break;
            int p=prime[i];
            valid[t*p]=1;
            if(t%p==0)
            {
                phi[t*p]=phi[t]*p;
                break;
            }
            else
            {
                phi[t*p]=phi[t]*phi[p];
            }
        }
    }
}
void solve()
{
    int ans=0;
    cin>>n;
    init();
    for(int t=1;t<=n;++t)
    {
        sum[t]=sum[t-1]+phi[t];
    }
    for(int t=1;t<=cnt;++t)
    {
        ans+=2ll*sum[n/prime[t]]-1;
    }
    cout<<ans;
}
signed main()
{
    T=1;
    // cin>>T;
    while(T--)
    {
        solve();
    }
    return 0;
}
P3601

https://www.luogu.com.cn/problem/P3601

$$ \sum_{i=l}^{r} \text{qiandao}(i)\mod 666623333 $$

注意到 $l,r$ 范围虽然很大，但是 $r-l$ 的值在可处理数据规模内，我们考虑求区间素数一样的线性筛思路。我们发现一个数的 $\phi$ 是由它的因数决定的，


#0x01 定理1

$phi(p)$ 是积性函数，满足对于 $gcd(p,q)=1$，$phi(pq)=phi(p)*phi(q)$

  • $phi(p)=p-1$，$p$ 是素数
  • $phi(p^k)=p^{k-1}phi(p)=pphi(p^{k-1})$，$p$ 可以是任何数

证明：共 $p^k$ 个数字，仅含质因子 $p$ 的数与其不互质，通过大减小得 $phi(p^k)=p^k - p^{k-1}$

这个定理说明了，可以通过线性筛预处理欧拉函数

#include <bits/stdc++.h>
#define int long long
#define debug(x) cout<<#x<<':'<<x<<' ';
#define N
using namespace std;
int T;
int modd=998244353;
vector<int> prime,num;
int qpow(int u1,int u2)
{
    int res=1;
    while(u2)
    {
        if(u2&1)
        {
            res=res*u1;
            res%=modd;
        }
        u2>>=1ll;
        u1*=u1;
        u1%=modd;
    }
    return res;
}
void init(int n)
{
    for(int t=2;t<=sqrt(n);++t)
    {
        if(n%t==0)
        {
            int u1=0;
            while(n%t==0)
            {
                ++u1;
                n/=t;
            }
            prime.push_back(t);
            num.push_back(u1);
        }
    }
    if(n>1) 
    {
        prime.push_back(n);
        num.push_back(1);
    }
    return ;
}
int getphi(int n)
{
    int res=n;
    for(int t=2;t<=sqrt(n);++t)
    {
        if(n%t==0)
        {
            res=res/t*(t-1);
            while(n%t==0) n/=t;
        }
    }
    if(n>1) res=res/n*(n-1);
    return res;
}
void solve()
{
    int a,b;
    cin>>a>>b;
    init(a);
    int ans=1;
    for(int t=0;t<prime.size();++t)
    {
        int p=prime[t],tot=num[t];
        ans=ans*((qpow(p,b*tot-1)*getphi(p))%modd);
        ans%=modd;
    }
    cout<<ans;
}
signed main()
{
    T=1;
    // cin>>T;
    while(T--)
    {
        solve();
    }
    return 0;
}

P4388

https://www.luogu.com.cn/problem/P4388

​ 对于这个问题，将其等价为从左下到右上的情况后，会首先考虑一个问题，怎么快速判断一个矩形有多少方块被经过。首先，经过举例子打表发现，若两边互质，则不会有格点被经过。所以我们可以将不互质的情况（即经过格点的情况），分解为一个个互质的图再累加起来。对于两边互质的情况，我们发现，因为不会经过格点，所以可以看作是只能进行上下左右移动的情况，所以经过的方格一定是 $n+m-1$ 。对于更一般的情况，$\gcd (n,m)=d$ ,经过的方格数为 $\sum =d*(\frac{n}{d}+\frac{m}{d}-1)=n+m-d=N$ 。

​ 题目中给定了 $N$ ，要求我们求出满足条件的 $(n,m)$ 个数。我们考虑枚举 $d$ ，虽然我们定义的 $d$ 是 $n,m$ 的因数，但是在上式中会发现 $d$ 也一定是 $N$ 的因数，所以我们枚举 $N$ 的因数 $d’$，然后再由 $n+m=N+d $ 枚举 $n,m$ ，并且满足 $\gcd (n,m)=d$ 即可。用式子表示就是在求：

$$ \sum _{d|N}\sum_{n=1}^{N} [\gcd(n,N+d-n)=d] $$

这时候需要对 $gcd$ 的性质非常有感觉，因为上式必须把一个其中的参数变为常量才方便套用欧拉函数，等价于：

$$ \sum _{d|N}\sum_{n=1}^{N} [\gcd(n,N+d)=d] $$$$ \sum _{d|N}\sum_{n'=1}^{\frac{N}{d}} [\gcd(n',N/d+1)=1],n'=n/d $$$$ \sum _{d|N}\phi(\frac{N}{d}+1) $$$$ \sum _{d|N}\phi(d+1) $$

此时我们发现结果出错了，这时候发现题目中 $(n,m)$ 与 $(m,n)$ 等价，特殊处理一下即可。


#0x02 定理2

设 $n$ 为正整数，那么

$$ n=\sum_{d|n} phi(d) $$

证明：存在 $n$ 个分数分别为 $1/n,2/n,…,n/n$ 化简这些分数得到新的 $n$ 个分数，它们分子分母互质，分母一共有 $n$ 的因数种，每个分母 $d$ 的分子都有 $phi(d)$ 个

这是个非常强大的公式，可以用于处理 $gcd$ 相关的问题。

$$ gcd(i,j)=\sum_{d|gcd(i,j)} phi(d)=\sum_{d|i \text 且 d|j} phi(d) $$

例题

P2398

https://www.luogu.com.cn/problem/P2398

$$ \sum_{i=1}^{n} \sum_{j=1}^{n} \gcd(i,j) $$

利用公式，将 $gcd$ 拆开：

$$ \sum_{i=1}^{n} \sum_{j=1}^{n}\sum_{d|i \text 且 d|j} phi(d) $$$$ \sum_{i=1}^{n} \sum_{j=1}^{n}\sum_{d} phi(d)*[d|i]*[d|j] $$

交换求和顺序得：

$$ \sum_{d}phi(d)\sum_{i=1}^{n}\sum_{j=1}^{n} [d|i]*[d|j] $$

拆开 $\sum$ 得到：

$$ \sum_{d}phi(d)\sum_{i=1}^{n}[d|i]*\sum_{j=1}^{n}[d|j] $$

由 $\sum_{i=1}^n [d|i]=\lfloor n/d \rfloor$ 得：

$$ \sum_{d}phi(d)*\lfloor n/d \rfloor*\lfloor n/d \rfloor $$

因此预处理 $phi$ 后遍历 $n$ 的因数 $d$ 即可在 $O(n)$ 时间解决。

P1390

https://www.luogu.com.cn/problem/P1390

$$ \sum_{i=1}^{n} \sum_{j=i+1}^{n} \gcd(i,j) $$

**解法1：**结合之前结论，由对称性可得：

$$ \sum_{i=1}^{n} \sum_{j=i+1}^{n} \gcd(i,j)=(\sum_{i=1}^{n} \sum_{j=i}^{n} \gcd(i,j)-\sum_{i=1}^n i)/2 $$

**解法2：**由欧拉函数定义可得：

$$ \sum_{i=1}^{n} \sum_{j=i+1}^{n} \gcd(i,j)=\sum_{d=1}^n d\sum_{i=1}^{n/d} \sum_{j=i+1}^{n/d} [\gcd(i/d,j/d)=1] $$

会发现 $\sum_{i=1}^{n/d} \sum_{j=i+1}^{n/d} [\gcd(i/d,j/d)=1]$ 实际上是在求 $[1,n/d]$ 中互质的对数（注意不包括 $1,1$ ）

求 $[1,n]$ 中互质的对数：

$$ S(n)=\sum_{i=1}^n phi(i) $$

故原式可化为：

$$ \sum_{d=1}^n d\sum_{i=1}^{n/d} \sum_{j=i+1}^{n/d} [\gcd(i/d,j/d)=1]=\sum_{d=1}^n d* (S(n/d)-S(1)) $$

P1891

https://www.luogu.com.cn/problem/P1891

$$ \sum_{i=1}^{n} \operatorname{lcm}(i,n) $$

将 $lcm$ 转化为 $gcd$ 问题：

$$ n*\sum_{i=1}^n i/gcd(i,n) $$$$ n*\sum_{d|n} \sum_{i=1}^n i/d*[gcd(i,n)=d] $$$$ n*\sum_{d|n} \sum_{k=1}^{m} k*[gcd(k,m)=1],k=i/d,m=n/d $$

设 $G(n)=\sum_{i=1}^n i*[gcd(i,n)=1]$，实际上这是互质数和函数，我们会发现一个数的所有互质数总是成对出现，若 $p$ 与 $n$ 的互质，则 $n-p$ 一定也与 $n$ 互质，每对数的和恒为 $n$ ，因此 $G(n)= n*phi(n)/2$ ，特别地，$G(1)=1$ 。因此，抛开 $1$ 的情况，原式可化简为：

$$ n*\sum_{d|n} \frac {n*phi(\frac{n}{d})}{2*d} $$

或者换成更优美的形式：

$$ n*\sum_{d|n} \frac {d*phi(d)}{2} $$

ll valid[N],phi[N];
void init(int n)
{
    phi[1]=1;
    for(int t=2;t<=n;++t)
    {

            {
                phi[t*p]=phi[t]*p;
                break;
            }
            else phi[t*p]=phi[t]*phi[p];
        }
    }
}
void solve()
{
    int n;
    // n=read();
    cin>>n;
    ll ans=0;
    for(int t=1;t<=sqrt(n);++t)
    {
        if(n%t==0)
        {
            int i=t;
            if(i==1) ans+=1ll;
            else ans+=i*phi[i]/2ll;

            if(t==n/t) continue;
            i=n/t;
            if(i==1) ans+=1ll;
            else ans+=i*phi[i]/2ll;

#0x03 欧拉定理

**欧拉定理：**设 $m$ 是一个正整数， $a$ 是一个整数且 $a$ 与 $m$ 互素，即 $\gcd(a,m)=1$，则有 $a^{\phi(m)} \equiv 1(\bmod m)$。

这个定理有以下两个应用点：

  • 此定理将同余式，或者说将取模运算，整除运算和欧拉函数联系在了一起。
  • 此定理可以用于处理未知量在指数的情况下，也就是将指数和底数与欧拉函数联系在一起。

例题

P10496

一个数字所有位数都是数字 $i$ 的数可以表示为：

$$ \frac{10^x-1}{9}*i $$

此题可以表示为找到最小的 $x$ 使其满足：

$$ L |\frac{10^x-1}{9}*8 $$$$ 9L |(10^x-1)*8 $$

转化为同余式：

$$ (10^x-1)*8 \equiv 0(\bmod 9L) $$

定义 $d=\gcd(9L,8)$

$$ 10^x \equiv 1(\bmod \frac{9L}{d}) $$

我们知道欧拉定理告诉我们，当且仅当 $\gcd(10,\frac{9L}{d})=1$时，等式成立，但是题目要求的是 $x$ 满足条件的最小值，其实可以感性理解以下，可能会存在更小的值 $x$ 满足这个式子，由积性函数的性质得，一定是 $\phi(\frac{9L}{d})$ 的因数。

	return b?ygcd(b,a%b):a;

int qpow(int u1,int u2,int modd)
{
    int res=1;
    while(u2)
    {
        if(u2&1)
        {
            res=res*u1;
            res%=modd;
        }
        u1*=u1;
        u1%=modd;
        u2>>=1ll;
    }
    return res;
}
int getphi(int n)
{
    int res=n;
    for(int t=2;t*t<=n;++t)
    {
        if(n%t==0)
        {
            res=res/t*(t-1);
            while(n%t==0) n/=t;
        }
    }
    if(n>1) res=res/n*(n-1);
    return res;
}

{
    int n,mod;
    for(int Y=1;;++Y)
    {
        n=read();
        if(n==0) break;
        n=9*n/ygcd(9*n,8);
        if(ygcd(10ll,n)!=1)
        {
            cout<<"Case ";
            print(Y);
            cout<<": "<<0<<'\n';
            continue;
        }
        
        int u1=getphi(n);
        int ans=u1;
        for(int t=1;t*t<=u1;++t)
        {
            int i=t;
            if(u1%i!=0) continue;

            if(qpow(10ll,i,n)==1)
            {
                ans=min(ans,i);
            }
            i=u1/t;
            if(qpow(10ll,i,n)==1)
            {
                ans=min(ans,i);
            }
        }
        cout<<"Case ";print(Y);
        cout<<": ";
        print(ans);
        cout<<'\n';
    }
}
